What You Will Learn
Learning Objectives

📌 By the end of this module, you will be able to:

  • Explain the basics of HTML, CSS, and JavaScript and understand how browsers execute code
  • Describe how Cross-Site Scripting (XSS) vulnerabilities arise from unsanitized user input
  • Identify the three types of XSS — Reflected, Stored, and DOM-based
  • Manually test a web application for XSS vulnerabilities using common payloads
  • Apply basic input sanitization and output encoding to fix vulnerabilities
